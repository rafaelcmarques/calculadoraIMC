## ⚖️ O que é IMC?

Indice de Massa Corporal, conhecido pela sigla IMC, é um cálculo simples que permite medir se alguém está ou não com o peso ideal. 
Ele aponta se o peso está adequado ou se está abaixo ou acima do peso.
